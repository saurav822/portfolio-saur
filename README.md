# Saurav Rajput — Portfolio

Personal portfolio site for Saurav Rajput, Senior Data Scientist & AI Builder.

**Live at:** [saurav822.github.io/portfolio-saur](https://saurav822.github.io/portfolio-saur)

## Stack

- Vanilla HTML/CSS/JS — no frameworks, no build step
- Fonts: Space Grotesk, Inter, JetBrains Mono (Google Fonts)
- Animations: CSS keyframes + IntersectionObserver scroll reveals

## Structure

```
index.html   — entire site (single file)
```

## Sections

- **Hero** — headline + typewriter role cycler
- **Trusted At** — IIT Delhi · IIM Calcutta · Walmart · BCG · eBay
- **Impact** — key numbers across engagements
- **Origin Story** — timeline from 2017 → now
- **Work** — selected projects ordered by impact
- **Build** — skills, tools, and a few lab experiments
- **Contact** — links and reach-out

## Running Locally

```bash
open index.html
# or
npx serve .
```
